5.4.3 Release Notes

Skip to end of metadata
  • Created by Bertrand Dunogier, last modified on Jul 29, 2015
Go to start of metadata
July 27th 2015

eZ announces the availability of 5.4.3, a maintenance release available for all users of eZ Publish Platform 5.4 containing a notable few updates and fixes.

Follow the 5.4.x Update Instructions to apply this update to your platforms.

Package updates
