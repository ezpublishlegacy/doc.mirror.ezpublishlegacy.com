These pages covers eZ Publish Platform 5.x releases, for Community releases see share.ez.no, for eZ Platform releases see Releases on TECHDOC space.

If upgrading from a previous version of eZ Publish Platform please pay attention to the Upgrade Guide of the latest version, it might contain upgrade guide for upgrading directly from your existing version to latest, otherwise you will have to follow the upgrade guide for each version of eZ Publish Platform release you are 'skipping'!
