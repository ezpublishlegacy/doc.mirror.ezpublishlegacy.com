Places Block allows you to create a list of places with their details.

 

The Places Block displays a list of all place Content items in a folder you choose.

The list will contain the name and photo thumbnail of the place. It will allow you to enter the place entry with further information about the place, including location on a map.

Choosing places to display

 

 

You can select places to display in the Universal Discovery Widget.

You need to select a folder with places, not a single place Content item, for the Places Block to work.
