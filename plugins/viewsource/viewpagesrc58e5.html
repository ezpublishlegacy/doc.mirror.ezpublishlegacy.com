Content List Block allows you to display a list of Content of a selected Type.

 

The Content Llist Block displays Content items of a chosen Content Type (or Types) that is contained in a folder. The layout of the Content items can be defined using templates.

Available Content List settings

 

OptionDescription
Name

Enter the Content List Block name.

Parent
Use the Universal Discovery Widget to select a folder.
Limit

Enter the maximum number of Content items which will be displayed.

Content Types to be displayed
Select Content Type(s) from the list.
