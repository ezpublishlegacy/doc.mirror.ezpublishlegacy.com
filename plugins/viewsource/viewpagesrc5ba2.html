You can view and edit any previously created version of a Content item.

eZ Enterprise tracks every time when you save a Content item you are working on. You can later, at any time, view a list of all previous versions of a Content item and view or edit any of them.

Page mode interface

 

 

Viewing or editing a previous version

 

1. Go to the Page mode and open a Content item you would like to view. 

2. In the Studio toolbar, click the Versions button:

A window appears with a list of all versions that the current Content item has.

3. Hover over the versions you want to see and click the view or edit button.
