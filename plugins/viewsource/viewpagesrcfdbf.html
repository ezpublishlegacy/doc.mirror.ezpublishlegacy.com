Currently the following versions are supported in the eZ Publish Cloud solution:

  • eZ Publish 4.7
  • eZ Publish 5.1 LS* ( Legacy Stack )

*LS means that that only legacy stack is supported. Running eZ Publish with Symfony is yet not supported.
