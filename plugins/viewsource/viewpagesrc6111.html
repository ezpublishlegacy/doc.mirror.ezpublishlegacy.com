The Admin Panel gives you access to administration options for the whole website, such as managing users, permissions, modifying Content Types and sections.

Admin Panel options

 

 

The Admin Panel offers the following tabs:

  • Administration dashboard - access all the administrative options from one hub
  • System information - view information about your current installation and server
  • Sections - manage the way your Content is organized into Sections
  • Content types - manage the Content Types set up in your system
  • Languages - manage the Languages set up in your system
  • Users - manage User accounts and User Groups
  • Roles - manage the permission system
