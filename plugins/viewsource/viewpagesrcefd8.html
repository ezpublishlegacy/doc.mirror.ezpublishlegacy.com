Known missing integration (black list):

FeatureUnsupported
DoctrineWe don't currently use Doctrine DBAL, options: use Zeta Database/Persistence until we switch or be ok with several database connections. If the latter you can use it like described in symfony doc by adding doctrine/orm & doctrine/doctrine-bundle via composer and installs it like any other bundle.
AnnotationsWe don't currently bundle FrameworkExtraBundle which provides support for Annotating routes/firewall rules on controllers and other features, workaround: customer can add it in composer and installs it like any other bundle.
