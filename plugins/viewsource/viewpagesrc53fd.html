Get started:

  • Become active in the community
    • share.ez.no
    • Slack
  • Comment on mistakes in the doc so others will be aware
    • We will make sure to clean up the comments once the reported issue has been fixed


Contribution coordination:

  • Documentation:
    • On Slack use #documentation-contrib
  • Code:
    • Make sure to sign a CLA: Contributor Licensing Agreement-CLA
    • On Slack use a relevant channel:
      • #ezplatform-core-dev
      • #ezplatform-ui-dev

 

And watch this space for future tutorials on how to contribute to the different parts of eZ Platform.
