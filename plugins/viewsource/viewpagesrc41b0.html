Keyword Block allows you to display all Content with selected keywords assigned.

 

The Keyword Block lets you specify a keyword or keywords. It will then display all Content items which contain the selected keywords.

 

Adding blocks to Landing Page

 

 

Available Keyword Block settings

 

OptionDescription
Name

Use it to define the Keyword Block name.

Keywords

Choose the keyword or keywords that will be used to select Content to display.

The block will display all Content items which contain at least one of the selected keywords.

Content Types to be displayedSelect Content Type(s) from the list.
Limit

Enter the maximum number of items that will be shown in the block.
