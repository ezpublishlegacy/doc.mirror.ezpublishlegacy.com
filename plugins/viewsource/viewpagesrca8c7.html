Problem

You want to get eZ Publish Platform on your server and can't find the software.

Solution

You can get eZ Publish Platform by 3 ways

:

  1. Downloading the Community Edition
  2. Cloning the git repository
  3. Downloading the Enterprise Edition

eZ Publish Community Edition

Note that eZ Systems has a distinction between

  • the enterprise edition of eZ Publish, which is the stable, commercial version of the product
  • and the Community Edition, a joint development project to cater for a product that stays in touch with the market trends.

Each 6 month new, relevant developments are tested and updated to the eZ Publish Enterprise.

The community edition and the eZ Publish extensions can be downloaded from our Community Portal.

Note : you can also clone the git repository of our ezpublish-kernel on Github and install dependencies with Composer.

eZ Publish Enterprise Edition

You can download an eZ Publish Enterprise trial version from ez.no.

Only customers having subscribed to the eZ Publish Enterprise Subscription get access to eZ Publish Enterprise. There, you will find all the details about the Subscription Services & Tools that help enterprises and organizations improve and secure their eZ Publish content platform.

You will need to submit the form to qualify for a trial agreement for a 30-days trial.
