Overview

The eZ content model is based on managing Content items. Each Content item is an instance of a specific Content Type, and contains a number of Fields.

 

See Fundamental conceptsFundamental conceptsto learn more about the concepts of the content model.

In this section of the manual you can read detailed information about elements of the eZ content model: Content Types and Field Types.
