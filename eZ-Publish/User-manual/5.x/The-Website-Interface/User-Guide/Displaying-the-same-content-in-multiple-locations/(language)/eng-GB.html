Displaying the same content in multiple locations

The same content object can be published to multiple locations within the content node tree. This enables a single content object to be displayed in more than one part of the website.

To publish a piece of content to an additional location, navigate to the desired page then click on the Locations button.

After clicking on the button, the top-level node of the content node tree will be displayed. Click on the highlighted container nodes to navigate the tree until you get to to the location where you want to publish the content. When you have found the new location, check the box and click on the Select button.
