IconFormat

Summary

Sets which suffix to use for flag icon files.

Usage

IconFormat=gif

Description

This setting sets which suffix to use for all flag icon files.This setting is case sensitive.

Examples

IconFormat=gif

This is the default setting.  All flag icons will have the suffix "gif", for example eng-GB.gif
