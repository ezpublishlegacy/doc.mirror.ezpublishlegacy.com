Statistics

Statistics

You can get brief statistical information about the eZ Recommendation services usage and results. To see this information in the backend, open the new ezrecommendation entry in the left row of the Setup tab.

 

eZ Recommender Statistics

It provides absolute numbers about:

1)    Click events that were triggered

2)    Purchase events that were triggered (online shop)

3)    Delivered recommendations (i.e. number of recommendation calls)

4)    Recommendations that were clicked by users

5)    Recommendations that were purchased

6)    Revenue that can be calculated from purchased recommendations and the price of each individual product if the shop solution is used

The following screenshots are examples that provide statistics for a publisher and a shop solution. Instead of consume events, you have three additional columns in the shop solution named ‘Revenue’, ‘Purchased-Recommended’ and ‘Purchase-Events’.
