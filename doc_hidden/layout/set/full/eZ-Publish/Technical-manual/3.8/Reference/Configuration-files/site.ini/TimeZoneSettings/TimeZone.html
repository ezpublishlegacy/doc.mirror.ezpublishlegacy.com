TimeZone

Summary

Sets the timezone to use in eZ publish.

Usage

 
TimeZone= Area/Location

Description

In order to run multiple installations of eZ publish on one server using different timezones it is now possible to override the OS default timezone. The time zones are named as "Area/Location", where "Area" is the name of a continent or ocean, and "Location" is the name of a specific location within that region, usually cities or small islands. (The list of timezones can be found at http://www.php.net/manual/en/timezones.php)

Examples

TimeZone=America/Antigua

This will instruct the system to use the "America/Antigua" timezone instead of the OS default timezone.
