ContentXMLCharset

Summary

Sets the characterset used when storing XML in content objects.

Usage

 
ContentXMLCharset=enabled|disabled| character_set

Description

  • enabled - Stores XML fields using the current character set. The current character set is set by the Charset setting. This setting is the correct setting for most people.
  • disabled - Forces storage in UTF8.
  • character set - Forces storage in this character set.
