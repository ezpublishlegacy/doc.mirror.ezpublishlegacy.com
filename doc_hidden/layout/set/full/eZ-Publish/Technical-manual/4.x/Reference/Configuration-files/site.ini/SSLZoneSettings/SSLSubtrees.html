SSLSubtrees

Summary

Content subtrees to use SSL on.

Usage

SSLSubtrees[]=/<subtree>

Description

Define the subtrees where you desire to use SSL. It expects the location to a subtree as value.

For this setting to be considered, the SSLZones setting must be set to enabled.

Examples

SSLSubtrees[]
SSLSubtrees[]=/news
SSLSubtrees[]=/polls
