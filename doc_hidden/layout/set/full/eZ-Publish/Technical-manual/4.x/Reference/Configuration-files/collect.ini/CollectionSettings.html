[CollectionSettings]

SettingSummary
CollectAnonymousData Sets if information from anonymous users can be collected.
CollectAnonymousDataAttribute Sets which attribute in the object determines if collecting anonymous data is permitted.
CollectAnonymousDataList Sets if information from anonymous users can be collected for specific classes.
CollectionUserData Sets how information collection is handled in terms of user identification
CollectionUserDataAttribute Sets which attribute in the object determines how information in collected.
CollectionUserDataList Sets how information collection is handled in terms of user identification for specific classes.
