AdditionalTemplates

Summary

A list of templates to include in the content/edit form.

Usage

AdditionalTemplates[]=content/edit/autosave.tpl

Description

This setting will include the template "design:content/edit/autosave.tpl" from "design:content/edit.tpl".

Note: This is relevant to the 4.7 version, and later versions, of eZ Publish.

Examples

Example: "content.ini.append.php"

[EditSettings]
AdditionalTemplates[]=content/edit/autosave.tpl
