Reference

Summary

This setting will tell the system which alias to use as reference image for "rss" images.

Usage

 
Reference=<alias>

Description

This setting will tell the system which alias to use as reference image for "rss" images. The reference image is the base of conversion and filtering.

This setting is case sensitive.

Examples

Reference=reference
