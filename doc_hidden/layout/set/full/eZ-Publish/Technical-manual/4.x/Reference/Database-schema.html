Database schema

Below please find a link to the eZ Publish database schema, the blueprint of the eZ Publish database. The database schema defines the tables of the eZ Publish database and the relationship between them.

The database schema documentation contains tabs that will show you the database tables from various "angles".
Please know that the schema consists of drill-down elements, so you can click down to the next level of information.
