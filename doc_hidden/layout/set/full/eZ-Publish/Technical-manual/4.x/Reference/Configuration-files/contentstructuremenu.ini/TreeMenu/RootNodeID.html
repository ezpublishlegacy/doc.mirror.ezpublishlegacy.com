RootNodeID

Summary

Sets the root node for the tree menu.

Usage

 
RootNodeID=node_id

Description

This setting controls which node to use as the root one for the tree menu. The default value is 2 which means that the "Content structure" menu will display a tree containing the nodes that belong to the "Content" top level node (node ID= 2).
