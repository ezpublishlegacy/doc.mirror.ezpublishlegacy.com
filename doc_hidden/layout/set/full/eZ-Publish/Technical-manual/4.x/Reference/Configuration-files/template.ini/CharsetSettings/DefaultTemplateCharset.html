DefaultTemplateCharset

Summary

Sets the default charset to use.

Usage

DefaultTemplateCharset=<charset_name>

Description

This setting sets which character set (charset) to use when no charset is specified in the template.The charset defined here will also be used by transform.ini to map charsets to a language group.This setting is case sensitive.

Examples

DefaultTemplateCharset=utf-8

This is the default setting.
