Utf8Encoding

Summary

Sets the use of LDAP encoding.

Usage

 
Utf8Encoding=true | false

Description

This setting allows you to enable or disable if LDAP encoding is utf-8 or not.

Possible values are true and false. True sets LDAP encoding to UTF-8.

This setting is case sensitive.

Examples

Utf8Encoding=false

UTF-8 LDAP encoding will not be used.
