TopIdentifierList

Summary

Lists the identifiers of the classses that are allowed to be shown in top menu's.

Usage

TopIdentifierList[]=<class identifier>

Description

This list contains the identifiers of the classses that are allowed to be shown in top menu's.

Note : from 5.3 / 2014.07 version the class listed here will appear in the Menu. This is implemented in the DemoBundle.

Examples

TopIdentifierList[]=folder
TopIdentifierList[]=feedback_form
TopIdentifierList[]=gallery
TopIdentifierList[]=forum

This is the default value used for this setting.
