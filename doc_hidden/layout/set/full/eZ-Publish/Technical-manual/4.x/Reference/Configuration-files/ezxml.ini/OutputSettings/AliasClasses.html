AliasClasses

Summary

sets which alias to use

Usage

 
AliasClasses[]=

Description

Since eZ Publish 4.1 this setting replaces the "Alias" configuration setting.

Examples

Example 1

AliasClasses[eZXHTMLXMLOutput]=eZPlainXMLOutput
