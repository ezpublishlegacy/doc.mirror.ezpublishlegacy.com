AppendLastModifiedTime

Summary

Appends the last modified time to the cached file name.

Usage

AppendLastModifiedTime=disabled

Description

This setting will append the last modified time to the cached file name, in order to force the Proxy/Browser cache to re-fetch the file when changed.

Keep in mind that the cronjob to cleanup expired files does not exist by default, but can be created. For more information on cronjobs please visit: http://ez.no/doc/ez_publish/technical_manual/4_x/features/cronjobs

This setting is case sensitive.
