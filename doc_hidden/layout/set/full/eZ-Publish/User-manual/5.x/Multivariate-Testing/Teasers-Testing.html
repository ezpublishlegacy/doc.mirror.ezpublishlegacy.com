Teasers Testing

This section covers teasers testing, a part of the eZ Publish multivariate testing feature.

Follow the steps below in order to setup your teaser testing scenario.

Note that it is done via dedicated Multivariate Test Container class.

1. Navigate over eZ Publish content tree and locate the content you want to test against.
2. Create a new multivariate test container object as shown below:

Multivariate Test Container Object

3. Give your container a name. In most cases it will be exactly the same as the content you are testing against, i.e.: an article title.

Test Container Name

4. On your test container, create the new content you want to test against, as shown below:

Test Container Content

5. Define the probability by entering proper values for Priority.

Note that the total value (percentage) amount for all test items cannot exceed 100%.

Test Container Probability

6. The container is responsible for displaying a proper content for the end user, depending on the probability.By default only line view is supported. Once testing is completed, simply move the object from the container. Then you can remove testing containers with unused test items.
