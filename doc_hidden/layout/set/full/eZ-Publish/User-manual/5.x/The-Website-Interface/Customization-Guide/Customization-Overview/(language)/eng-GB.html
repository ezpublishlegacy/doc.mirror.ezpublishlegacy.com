Customization Overview

There are two methods for customizing an eZ Publish Website Interface site:

  • Creating a site style package
  • Modifying the default CSS

You can modify basic design elements (such as colors, fonts, etc) via site style packages.This manual describes both how to create a site style package and how to modify thedefault design.
