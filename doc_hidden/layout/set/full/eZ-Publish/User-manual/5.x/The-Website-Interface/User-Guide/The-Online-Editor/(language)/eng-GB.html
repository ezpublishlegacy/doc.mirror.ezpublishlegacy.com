The Online Editor

The Online Editor is a web-based rich text editor that enables you to do advanced text formatting (such as headings, tables, lists and links) and to embed content objects into text blocks. When you  edit a content object, any attributes of the content object that allow rich formatting are displayed within the Online Editor interface.
