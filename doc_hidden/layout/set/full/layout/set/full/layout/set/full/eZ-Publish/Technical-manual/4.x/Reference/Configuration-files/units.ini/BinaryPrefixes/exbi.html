exbi

Summary

Sets the base unit for exabyte.

Usage

 
exbi=<factor>;<symbol>

Description

This setting sets the factor value and the symbol for an array to use for exabyte. The factor used for calculating values in the power of 2.
This setting is case sensitive.

Examples

exbi=60;Ei

This is the default setting, indicating that the base unit for kilobyte is "Ei" and the factor is 2 60.
