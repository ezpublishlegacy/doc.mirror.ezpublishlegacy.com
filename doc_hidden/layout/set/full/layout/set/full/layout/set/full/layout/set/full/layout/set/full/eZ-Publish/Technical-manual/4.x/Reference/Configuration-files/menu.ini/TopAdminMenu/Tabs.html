Tabs

Summary

Lists the menu items of the top menu in administrator interface.

Usage

 
Tabs[]=<menu name>

Description

This list contains menu items (or tabs) of the top menu in administrator interface. This setting is case sensitive.

Note: As of eZ Publish 4.3 the tabs "shop" and "design" are hidden by default. To enable the "shop" and "design" follow the example below.

Examples

Tabs[]=dashboard
 Tabs[]=content
 Tabs[]=media
 Tabs[]=users
 Tabs[]=setup

The default setting is that the dashboard, content, media, users and setup tabs are displayed.

You enable the eZ Webshop tab "shop" and the design tab "design" by entering the following settings:

Tabs[]=shop
Tabs[]=design
