Timeline Preview

Because the contents of an eZ Flow front page change over time, eZ Flow provides a time-line preview that enables you to view the visitorâ€™s view of a front page at any point in time.

To enable the time-line preview, click the button on the Website Toolbar (marked in red on the next screenshot):

The time-line preview will then be displayed at the top of the page.

To select a specific time, drag the slider to the desired position. The page will then automatically be updated with the content that is scheduled to be published at that particular time. The slider allows selection within a 24 hour period.

If you wish to preview outside of the 24 hour scale provided by the slider, click the date selection drop-down and choose the desired date from the calendar.

Please note: If you change the time setting of your server and change the time zone of your Apache-PHP web server in the php.ini file, the eZ Flow time-line system might be impacted. The reload of the front page when moving the slider of the time-line might not work for a period of time corresponding to the jump you made in changing your time zone.

For instance moving from Chicago to Seattle (two hour time difference) might cause the time-line slider to not work properly for two hours
