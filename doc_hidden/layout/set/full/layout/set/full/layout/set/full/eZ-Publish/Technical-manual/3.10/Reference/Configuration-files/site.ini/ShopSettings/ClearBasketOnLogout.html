ClearBasketOnLogout

Summary

Sets whether the basket should be cleared after logging out.

Usage

 
ClearBasketOnLogout=enabled|disabled

Description

This setting has two possible values:

  1. disabled - Means that the basket is NOT cleared when the user logs out.
  2. enabled - Means to clear the basket as soon as the user logs out from the system.

Both anonymous and registered users are affected.

Examples

ClearBasketOnLogout=enabled

This will instruct eZ publish to clear the user's basket as soon as the user logs out from the system.
