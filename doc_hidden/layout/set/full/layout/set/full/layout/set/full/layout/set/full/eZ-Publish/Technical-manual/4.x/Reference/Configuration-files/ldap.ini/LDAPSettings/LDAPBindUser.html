LDAPBindUser

Summary

Sets the user name for LDAP server.

Usage

 
LDAPBindUser=<user name>

Description

If the server does not allow anonymous bind, specify the username for the bind here. A bind is a LDAP term for authenticating against a server, an anonymous bind is a bind without username, password or DN.
Note that the bind user has nothing to do with the credentials of the user logging in. Instead, it is the username that must be supplied in order to read the LDAP user object we want to authenticate against.

This setting is case sensitive.

Examples

LDAPBindUser=admin
