PointSize

Summary

This setting defines the point size for the specific style defined by the configuration block. Usage

Usage

PointSize=<integer>

Description

This setting defines the point size for the specific style defined by the configuration block.  The value to use is the number of pixels.

Examples

[gallery]
Family=bluehigc
PointSize=18
XAdjustment=0
YAdjustment=0
WidthAdjustment=8
HeightAdjustment=6
BackgroundColor=#ffffff
TextColor=#000000

The style gallery will use a point size of 18 pixels.
