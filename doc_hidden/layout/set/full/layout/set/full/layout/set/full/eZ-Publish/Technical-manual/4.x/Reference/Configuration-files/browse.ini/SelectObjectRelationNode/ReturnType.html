ReturnType

Summary

This setting defines the type of the return values.

Usage

 
ReturnType=NodeID | ObjectID

Description

This setting defines the type of the return values.
Possible values are:

  • NodeID: Returns the node ids of the selected items
  • ObjectID: Returns the object ids of the selected items

This setting is case sensitive.

Examples

ReturnType=NodeID

"NodeID" is the default value for this setting.
