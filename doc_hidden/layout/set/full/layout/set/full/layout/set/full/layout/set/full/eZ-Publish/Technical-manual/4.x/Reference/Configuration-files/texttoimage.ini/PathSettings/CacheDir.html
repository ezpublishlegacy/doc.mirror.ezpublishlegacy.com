CacheDir

Summary

Sets the location of the folder that contains the relevant cache files.

Usage

CacheDir=<path_to_directory>

Description

This setting defines the location of the folder that contains the cache files of the text to image.

These cache files are used when the setting [ImageSettings] UseChache is set to enabled.

This setting is case sensitive.

Examples

&nbsp;CacheDir=var/cache/texttoimage

This is the default location.
