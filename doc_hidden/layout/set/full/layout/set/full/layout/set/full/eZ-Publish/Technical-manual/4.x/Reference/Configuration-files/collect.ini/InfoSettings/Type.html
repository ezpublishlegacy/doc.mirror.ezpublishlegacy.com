Type

Summary

Sets the default information collection type.

Usage

 
Type=<collection type>

Description

This setting sets the default information collection type to use. An override can be set with the setting TypeList in the block [InfoSettings].
By default the following values are possible:

  • poll
  • feedback
  • form: the default value

All values have their own template located here:
<eZ Publish installation>\templates\content\collectedinfo
This setting is case sensitive.

Examples

Type=form

This is the default setting.
