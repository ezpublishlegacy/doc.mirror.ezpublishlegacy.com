ForceVirtualHost

Summary

Sets if eZ Publish should force virtual host mode.

Usage

ForceVirtualHost=true|false

Description

Normally eZ Publish automatically detects if virtual host or non virtualhost should be used. This has been further improved in 4.5, however some special cases require virtual host mode to be forced.

You can read more about the various access methods for eZ Publish in the Access Methods chapter. Specific information about virtual host setups can be found in the Virtual host setup chapter.
