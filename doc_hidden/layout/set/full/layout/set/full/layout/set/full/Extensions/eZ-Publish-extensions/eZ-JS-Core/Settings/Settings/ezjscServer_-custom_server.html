[ezjscServer_⎨custom_server⎬]

The settings defined here are supported by ezjscPacker and the classes used are ezcss and ezscript.

Examples are:

  •  [ezjscServer_ezjsc]
     The URL to test this server function (returns the current time) is the following:
    <root>/ezjscore/call/ezjsc::time
  •  [ezjscServer_ezjscnode]
     The URL to test this server function (returns the node list) is the following:
    <root>/ezjscore/call/ezjscnode::subtree::2
  •  [ezjscServer_ezjsctemplate]
     The URL to test this server function (returns an alert message) is the following: 
    <root>/ezjscore/call/ezjsctemplate::alert

SettingSummary
ClassName Sets the class name.
File Sets the file to use.
Function This setting lists the functions the user needs to have access to.
PermissionPrFunction Sets permissions per function.
TemplateFunction Sets if a template is to be executed instead of a php class function.
