UseSIUnits

Summary

Sets if eZ publish should use OSI or SI prefixes for binary numbers

Usage

 
UseSIUnits=true|false

Description

If this setting is set to true eZ publish will use SI prefixes for binary numbers. If the setting is set to false eZ publish uses OSI prefixes for binary numbers. You can set the units that eZ publish should consider binary with the BinaryUnits setting.
