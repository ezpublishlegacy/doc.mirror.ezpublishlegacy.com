AnalyzerMIMEList

Summary

Sets formats which need extra analysis.

Usage

 
AnalyzerMIMEList[]=<MIME type>

Description

Lists formats which need extra analysis.

This setting is case sensitive.

Examples

AnalyzerMIMEList[]=image/gif

This sets that GIF images must be analyzed to check for animated GIFs, since animated GIFs are hard to convert to other formats.
