HandlerList

Summary

Lists the package installation handlers.

Usage

HandlerList[]=<name_handler>

Description

This setting lists the package installation handlers. Each extension which has package installation handlers support must place their package installation handlers in the sub-directory package installers located here:

...(root of your eZ Publish installation)/extension/<extension_name>/packageinstallers

This setting is case sensitive.

Examples

HandlerList[]
HandlerList[]=ezcontentobject

These are the default settings.
