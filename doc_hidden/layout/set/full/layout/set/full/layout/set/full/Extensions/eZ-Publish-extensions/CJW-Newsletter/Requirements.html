Requirements

The CJW Newsletter extension has the following requirements:

  • eZ Publish 4.x
  • eZ Components version 2009.2.1
  • PHP 5 and PHP 5 cli version with the ability to send out emails.

When the system requirements are met you can download the CJW Newsletter extension from the eZ projects website.  To install the extension follow the steps listed in the chapter "Installation."
