eZ Lightbox

A lightbox is, simply put, a kind of container in which you can place other items such as content objects, content nodes or even objects from external systems. It is a user related system, where each user can create lightboxes, add items to it and even have several lightboxes at the same time. Also, users can share a lightbox in order to be able to discuss its contents with other users.

The lightbox extension offers different features to extend it's functionality. For example it is possible to create custom lightbox item classes to support functions when adding items different from the existing content objects or content nodes. A lightbox can also be used to bookmark items or function as a shopping basket or wishlist.

Visit the use page to learn how to use a lightbox in the eZ Teamroom context.
