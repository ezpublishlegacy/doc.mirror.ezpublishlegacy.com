Daily tasks

The purpose of this chapter is to explain how daily tasks can be carried out using the eZ Publish administration interface. It is written for non-technical people who are simply using the system to manage content.

The following topics are covered:

  • Changing your user account
  • Navigating the node tree
  • Adding content
  • Editing content
  • Previewing content
  • Undoing changes
  • Dealing with edit conflicts
  • Publishing at multiple locations
  • Copying content
  • Moving content
  • Removing content
  • Restoring content
  • Hiding and revealing content
  • Using bookmarks
  • Swapping nodes
  • Managing users
  • Managing sections
  • Creating a protected area
  • Managing sub-items
