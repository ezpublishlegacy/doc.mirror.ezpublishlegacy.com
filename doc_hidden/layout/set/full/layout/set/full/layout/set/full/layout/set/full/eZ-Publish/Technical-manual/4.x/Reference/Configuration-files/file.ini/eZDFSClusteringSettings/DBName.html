DBName

Summary

This sets which database to use when clustering with the eZ DB File Handler.

Usage

 
DBName=<name database>

Description

This sets which database to use when clustering with the eZDB File Handler.

Use the actual database name as value. In most cases this value will be the same as the "Database" setting specified under the [DatabaseSettings] block of your "site.ini.append.php" configuration file.
When using an eZ DFS File Handler please refer to the settings in file.ini [eZDFSClusteringSettings].

By default this is set to "cluster".

This setting is case sensitive.

Examples

DBName=cluster
