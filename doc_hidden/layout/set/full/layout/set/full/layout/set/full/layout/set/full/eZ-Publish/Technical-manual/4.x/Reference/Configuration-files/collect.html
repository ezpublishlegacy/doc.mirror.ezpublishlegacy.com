collect.ini

[CollectionSettings]

SettingSummary
CollectAnonymousData Sets if information from anonymous users can be collected.
CollectAnonymousDataAttribute Sets which attribute in the object determines if collecting anonymous data is permitted.
CollectAnonymousDataList Sets if information from anonymous users can be collected for specific classes.
CollectionUserData Sets how information collection is handled in terms of user identification
CollectionUserDataAttribute Sets which attribute in the object determines how information in collected.
CollectionUserDataList Sets how information collection is handled in terms of user identification for specific classes.

[DisplaySettings]

SettingSummary
Display This setting will set which display to show after information is collected.
DisplayAttribute Sets which attribute of the object determines the display page.
DisplayList Sets which display to show after information is collected for specific classes
RedirectURL Sets the URL to redirect to.
RedirectURLAttribute Sets which attribute in the object determines the URL to redirect to.
RedirectURLList Sets the URL to redirect to for specific classes.

[EmailSettings]

SettingSummary
SendEmail Sets if the installation is allowed to send out emails.
SendEmailAttribute Sets which attribute in the object determines if an email should be sent.
SendEmailList Sets if the installation is allowed to send out emails for specific classes.

[InfoSettings]

SettingSummary
Type Sets the default information collection type.
TypeAttribute Sets which attribute in an object determines the information collection type.
TypeList Sets the information collection type for specific class ID's or identifiers.
