MountPointPath

Summary

This setting defines the path to the NFS Mount Point, which should be used when clustering with an eZ DFS File Handler.

Usage

 
MountPointPath=<path to NFS mount point>

Description

Since the eZ DFS File Handler is based on NFS, each eZ Publish installation sharing the same relational database must use the same cluster database.

Also each eZ Publish installation should have a local mount point to the same NFS export. The NFS server has to be available and writeable by the webserver's user on each eZ Publish server.

Also it recommended that each eZ Publish server is configured in the exact same way. Refer to your system and server manual on how to configure this for your system.

The NFS Mount Point is a local folder on each eZ Publish server that links to the network file system where the handler stores the files. The location of the local mount point must be set with this setting and it can be relative or absolute.

This setting is case sensitive.

Examples

MountPointPath=var/nfsmount
