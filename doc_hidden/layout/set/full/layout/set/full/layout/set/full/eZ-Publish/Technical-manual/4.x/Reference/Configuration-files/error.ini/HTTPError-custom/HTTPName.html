HTTPName

Summary

Definition for a custom HTTP error code.

Usage

HTTPName=<string>

Description

This configuration defines the text message to show for a given HTTP error code.

Note: The HTTPName setting must contain the correct string for the specific HTTP error code on its value.

Examples

Example for HTTP error code 429:
HTTPName=Too Many Requests
