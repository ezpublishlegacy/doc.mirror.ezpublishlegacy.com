LibrarySummary
ezdb Provides a database abstraction layer.

ezfile Provides cross platform file and compression utilities.

ezutils Provides small utilities (ini file parser, MIME type handlers, E-mail handling, etc.).
ezwebdav Provides a low level communication interface for the WebDAV protocol.
ezxml Provides low level XML DOM parsing and construction utilities (DEPRECATED).
