Q: How does the system provide the ability to rewrite URLs to provide shorter or friendlier URLs?

A: eZ Publish uses friendly URLs by default.
Some examples of live URLs:
http://ez.no/eZPublish
http://www.thebanker.com/Profiles
You can additionally set up manual URL aliases for campaigns etc. In the case of multiple URLs to the same content eZ Publish has support for canonical links so that the site keeps optimal search engine ranking.
