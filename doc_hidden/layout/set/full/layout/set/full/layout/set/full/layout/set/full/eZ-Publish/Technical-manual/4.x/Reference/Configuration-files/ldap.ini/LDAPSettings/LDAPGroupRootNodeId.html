LDAPGroupRootNodeId

Summary

This sets the group mapping settings.

Usage

 
LDAPGroupRootNodeId=<node id>

Description

This sets the group mapping settings. This setting sets the location of groups created by the LDAP login handler. The location must be the node ID of the group node that should be the parent of all created groups.
If this setting is left blank the root node ID where the LDAP groups are created, node id 5, will be used.
This setting is case sensitive.

Examples

LDAPGroupRootNodeId=5

The groups created by the LDAP login handler will be located at node 5.
This is the default setting.
