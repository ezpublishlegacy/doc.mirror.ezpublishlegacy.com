win32_Executable

Summary

Sets the name for the image executable for win32.

Usage

 
win32_Executable=<name executable>

Description

Sets the name for the image executable for win32.
If there is more than one executable, use ";" to separate them. An example of syntax in such a case is:

unix_Executable=<executable>;<executable>

 

This setting is case sensitive.

Examples

 

win32_Executable=convertim.exe;convert.exe


This is the default setting.
