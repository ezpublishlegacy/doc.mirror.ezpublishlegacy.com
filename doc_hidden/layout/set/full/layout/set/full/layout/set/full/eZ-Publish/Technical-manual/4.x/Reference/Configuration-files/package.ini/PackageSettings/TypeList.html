TypeList

Summary

Sets the package types for specific package ID's or identifiers.

Usage

TypeList[<ID>]=<package type>
TypeList[<Identifier>]=<package type>

Description

This setting list the package types and matches them to ID's and Identifiers.

This setting is case sensitive.

Examples

TypeList[]
 TypeList[contentobject]=Content object
 TypeList[contentclass]=Content class
 TypeList[sitestyle]=Site style
 TypeList[site]=Site

These are the default settings.
