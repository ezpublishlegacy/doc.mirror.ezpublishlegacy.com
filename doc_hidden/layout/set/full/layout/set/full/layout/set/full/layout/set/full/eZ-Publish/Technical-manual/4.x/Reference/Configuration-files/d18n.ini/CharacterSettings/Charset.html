Charset

Summary

Sets the character set that eZ Publish should use

Usage

 
Charset=character_set

Description

This directive can be used to set the internal charset for the site. The default value is "utf-8". Note that eZ Publish 4 requires UTF-8 to be used as the character set in the database, and thus you should never change this setting.

Examples

Example 1

 

Charset=utf-8


This will set the output character set of eZ Publish to UTF-8 (default).
