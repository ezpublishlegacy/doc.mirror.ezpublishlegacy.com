Require

Summary

Sets the test requirements.

Usage

 
Require=one | all

Description

Some database extensions are required for eZ Publish to work correctly. These extensions are listed in the setting "Extensions" in the same block. During the setup, tests are preformed to check if the extensions listed exist. This setting defines whether the existence of only one of these extensions is required for the test to succeed, or whether they all should exist.
Possible values are:

  • one: only one of the extensions listed must exist for the test to be successful.
  • all: all of the extensions listed must exist for the test to be successful.

This setting is case sensitive.

Examples

Require=one

Only one of the listed extensions in the setting "Extensions" in the same block are required for the setup test to be a success.
This is the default setting
