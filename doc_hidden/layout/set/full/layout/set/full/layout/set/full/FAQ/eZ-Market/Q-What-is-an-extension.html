Q: What is an extension?

A: An 'extension' refers to any additional eZ Publish module that extends the feature scope of eZ Publish using APIs, eZ Publish content or by integrating with external systems. eZ Publish extensions are available through the following channels:

  • The commercial extensions provided by our partners for sale on eZ Market, the eZ Systems application store.
  • The official, bundled extensions provided with each version of eZ Publish, downloadable from our Community Portal.
  • The community extensions available from the "Community Contribution Pages"
