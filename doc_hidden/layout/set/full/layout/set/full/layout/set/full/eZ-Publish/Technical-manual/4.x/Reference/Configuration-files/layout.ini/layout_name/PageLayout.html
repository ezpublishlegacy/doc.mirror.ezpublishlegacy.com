PageLayout

Summary

Sets the pagelayout to use for a specific layout.

Usage

PageLayout=<your page layout template>

Description

Specifies the page layout template file that will be used for a specific layout.

Examples

[fullscreen]
PageLayout=fullscreen_pagelayout.tpl
