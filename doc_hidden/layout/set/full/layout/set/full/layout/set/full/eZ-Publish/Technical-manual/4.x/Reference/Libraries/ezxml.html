ezxml

In previous versions of eZ Publish, this library provided low level XML DOM parsing and construction utilities. It has been deprecated in eZ Publish 4. Instead, the system makes use of the DOM functions in PHP 5 (refer to "Requirements for doing a normal installation " for more information).
