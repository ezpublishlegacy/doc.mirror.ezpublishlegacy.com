PermissionPrFunction

Summary

Sets permissions per function.

Usage

 PermissionPrFunction=enabled | disabled

Description

This is an optional setting which will enabled or disable permissions set per function. If it is set per function, then the function name will be appended to the function list name according to the following syntax:

<FunctionList_name>_<funtion_name>

If this name is not set in the FunctionList setting, a warning will be given. This setting is disabled by default.

This setting is case sensitive.
