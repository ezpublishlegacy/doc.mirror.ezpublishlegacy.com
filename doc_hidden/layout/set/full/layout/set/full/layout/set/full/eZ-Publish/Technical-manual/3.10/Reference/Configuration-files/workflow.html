[EventSettings]

SettingSummary
AvailableEventTypes Sets the name of the event types that eZ publish should search for.
ExtensionDirectories Sets the extensions that contain event types.
RepositoryDirectories Sets the directories where eZ publish will search for event types.

[OperationSettings]

SettingSummary
AvailableOperations Sets the triggers that are avaible from the administration interface.

[SimpleShippingWorkflow]

SettingSummary
ShippingCost Sets the cost of the shipping.
ShippingDescription Sets the description that will appear on the order when using the simle shipping workflow.
