FilePath

Summary

Sets the file location.

Usage

FilePath=<file path>

Description

This setting sets the location where to find the text file defined with the setting FileName.  A value could be any directory of your eZ Publish installation root directory.If the FilePath is root, the file will need to be saved under the eZ Publish root directory.

This setting is case sensitive.

Examples

Example 1

FilePath=root

This is the default setting. The file will be located under the root directory of your eZ Publish installation.
Example 2

FilePath=var

The file will be located under the var directory of your eZ Publish installation.
