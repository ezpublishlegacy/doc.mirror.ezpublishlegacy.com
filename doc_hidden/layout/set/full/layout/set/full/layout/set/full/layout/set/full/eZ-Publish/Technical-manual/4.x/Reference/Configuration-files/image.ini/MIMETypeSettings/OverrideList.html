OverrideList

Summary

Sets the override settings for MIME types.

Usage

 
OverrideList[]=<name setting>

Description

A list of settings which override certain MIME types. The overrides can define allowed and disallowed filters as well as extra filters.

This setting is case sensitive.

Examples

OverrideList[]=AnimatedGIF
 
OverrideList[]=LayeredPhotoshop
