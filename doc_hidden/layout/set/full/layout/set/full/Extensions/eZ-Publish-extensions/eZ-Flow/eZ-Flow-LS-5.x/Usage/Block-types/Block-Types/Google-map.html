Google map

Sometimes it can be useful to add a map to your front page and the Google Map Block enables this. This feature will display a location on a map embedded in a static block or a list of content in a map in a dynamic block.

To add a static block log in to the eZ Flow front-end editing interface to edit the front page. Select a zone and in the block type drop down list add "Google Map".

The following block will appear where you can specify a block name, map location and a key.

Instead of a static map block, you can also add several maps or Geo articles to your front page with the Google Map Items Block. To add this dynamic map block log in to the eZ Flow front-end editing interface. Select a zone and under Block type add Google Map Items block.

The following block will appear, where you are able to choose a source (which will limit the location where maps should be fetched) and define the additional parameters such as class, overflow and the limit of content you wish to display.

 
 Click "Send for publishing" to complete the procedure.
