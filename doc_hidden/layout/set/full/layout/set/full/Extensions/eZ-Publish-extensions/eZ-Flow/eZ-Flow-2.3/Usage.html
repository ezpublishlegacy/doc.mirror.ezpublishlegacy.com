Usage

This chapter will provide you with all the information needed to understand and use all eZ Flow features. Amongst other things, eZ Flow terms like zones, blocks, timeline preview and overflow are described and illustrated. How to use your iPhone to access an eZ Flow portal web-site is also explained.

But before anything else it's useful to get familiar with the eZ Flow front-end editing interface toolbars.
