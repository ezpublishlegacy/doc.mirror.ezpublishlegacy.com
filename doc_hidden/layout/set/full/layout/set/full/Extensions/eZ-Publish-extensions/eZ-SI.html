eZ SI

The eZ SI extension offers native support in eZ Publish for Edge Side Include (ESI), enabling faster page delivery, higher traffic and higher content refresh rate. ESI is a simple markup language used for dynamic Webpage assembly and delivery, making it possible to assemble pages from blocs requiring different refresh rates and update frequencies. eZ SI makes it possible for an eZ Publish site to generate ESI or SSI tags in its HTML page, with faster development and increased performance as direct result.

The eZ SI extension is intended for developers who need to leverage the ESI standard, combining eZ Publish with compatibles delivery technology such as Varnish http caching system or Akamai Content Delivery Network. This extension is typically used by Digital Media portals and their home pages.
