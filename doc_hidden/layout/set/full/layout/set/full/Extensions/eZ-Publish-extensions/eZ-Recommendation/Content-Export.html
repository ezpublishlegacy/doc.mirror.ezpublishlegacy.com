Content Export

The capability of content export for recommendable content has been enabled in the class definition.

To this get into effect we must establish an incremental content export that is triggered with each publishing activity and we must initiate a one-time export of the whole content when the export conditions of a class are changed within the class definition (e.g. you decide to export tags for all articles for recommendation and therefore change the class definition).
