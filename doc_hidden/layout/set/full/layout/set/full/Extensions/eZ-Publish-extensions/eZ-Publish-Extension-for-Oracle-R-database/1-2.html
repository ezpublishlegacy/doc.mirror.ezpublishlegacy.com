1.5

The eZ Publish Extension for Oracle® Database makes it possible to use Oracle as a database server for eZ Publish. Version 1.5 of the extension is a bug fix release, with the same requirements, and installation and upgrade procedures as version 1.4.

Refer to the documentation of the eZ Publish Extension for Oracle® Database version 1.4 for the installation manual and upgrade instructions.
