Multimedia carousel

This block allows you to show a presentation of hand picked content. It is available for articles, images and video. The following screenshot shows an example of such a carousel.

Use the arrows in the upper right corner of the block to scroll the carousel.

To add Multimedia Carousel block to your front page, log in to the eZ Flow front-end editing interface to edit the front page. Select a zone and select Multimedia Carousel in the block type drop down list.

The following block will be added to the block list, where you are able to name the block. Next choose a source to limit the location where the content should be fetched and define the additional parameters such as the class (e.g Image), overflow, and the rotation frequency of the content, set in minutes, hours or days. And don't forget to add the content, you wish to show in your carousel:

The images shown in blue are visible in the carousel, the others are not.

Click "Send for publishing" to complete the procedure.
