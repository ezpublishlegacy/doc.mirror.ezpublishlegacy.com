Requirements

  • eZ Publish 5.x (use same version number for eZ Publish and eZ Flow LS) installed on legacy mode. To ensure that your web server meets the proper requirements visit the eZ Publish installation manual.
  • Requirements for working with the Website Interface version 5.0, 5.1 or 5.2 (use same version number for eZ Publish and eZ Flow LS) are the same as those for the Administration Interface. The Online Editor (present in both the Website Interface and the Administration Interface) has a more specific set of requirements.
