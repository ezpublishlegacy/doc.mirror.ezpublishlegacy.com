Image Editor

The eZ Image Editor LS 5.0.0 extension is an image editor for simple image modifications integrated in the editing interface of any eZ Publish Content Object that has at least an image as attribute. It allows cropping, flipping, rotating and watermarking an image as well as modifying some of its attributes such as the contrast or brightness and many more.

eZ Image Editor LS 5.0.0 is bundled in eZ Publish version 5.0.

Note: eZ Image Editor supports the same file types as currently supported by ez Publish, .png, .jpg and .gif files.
