HandlerClass

Summary

sets which output handler to use

Usage

 
HandlerClass=

Description

Since eZ Publish 4.1 this setting replaces the "Handler" configuration setting.

This setting allows you to set which output handler to use. As value it takes a PHP class name as the output handler. The default setting is eZXHTMLXMLOutput. eZPlainXMLOutput can be used to see raw XML data.

Examples

Example 1

HandlerClass=eZPlainXMLOutput
