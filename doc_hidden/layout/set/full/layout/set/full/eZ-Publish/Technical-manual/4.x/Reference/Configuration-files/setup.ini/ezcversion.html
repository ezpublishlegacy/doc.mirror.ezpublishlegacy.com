[ezcversion]

The settings in this block are used to check for the correct eZ Components version by checking the existence of class and function. At the moment it isn't possible to get version information directly, because eZ Components does not have an API to get version information.

This configuration group was added to provide additional tests for the setup wizard.

SettingSummary
MinimumVersion Sets the minimum eZ Components version required.
TestClass Tests the eZ Components class during setup.
TestFunction Testing the eZ Components function during setup.
