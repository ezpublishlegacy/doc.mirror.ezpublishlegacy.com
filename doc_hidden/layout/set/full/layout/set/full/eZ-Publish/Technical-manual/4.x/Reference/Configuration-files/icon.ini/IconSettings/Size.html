Size

Summary

Sets the default icon size.

Usage

Size=<string>

Description

This setting sets the default icon size. The value here is a string, not the actual hight and width.  The actual sizes are defined with the setting [IconSettings] Sizes[]=.

This setting is case sensitive.

Examples

Size=normal

This is the default setting.
