CookieSecure

Summary

This setting sets whether or not cookies are allowed to be sent over an unsecure connection.

Usage

 
CookieSecure= false | true

Description

This is an optional settings for the override php.ini session cookie settings. The setting CookieSecure sets whether or not cookies are allowed to be sent over an unsecure connection.

If set to "true" the setting CookieSecure will only allow cookies to be sent over a secure connection.

Possible values are "true" to enable the secure function and "false" to disable the secure function. False is the default setting.

If this setting is undefined, the setting will be used as defined in the php.ini setting session.cookie_secure. For more information on all other session related setting in php.ini please visit http://www.php.net/manual/en/session.configuration.php.

This setting is case sensitive.

This setting is available for eZ Publish 4.4 and above.

Examples

CookieSecure=false

This is the default setting.
