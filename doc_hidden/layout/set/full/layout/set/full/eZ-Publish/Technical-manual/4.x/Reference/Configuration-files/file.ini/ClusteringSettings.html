[ClusteringSettings]

Contains settings required when clustering

SettingSummary
DBChunkSize This determines the byte size of the blocks into which files are split when they are fetched and inserted from the database, when clustering with the eZDB File Handler.
ExtensionDirectories DEPRECATED since eZ Publish 4.1.
FileHandler This setting sets the cluster file handler.
NonExistentStaleCacheHandling This setting defines what happens when a requested cache file is already being generated and no expired cache file exists.
