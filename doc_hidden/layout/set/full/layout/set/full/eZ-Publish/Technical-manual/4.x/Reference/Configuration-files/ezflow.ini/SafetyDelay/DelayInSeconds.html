DelayInSeconds

Summary

Sets a safety delay for publishing content

Usage

DelayInSeconds=<integer>

Description

This setting allows to tweak the safety delay used for content publishing, on eZ Flow only.
The expected value is an integer, which represents the amount of seconds that will be considered.

For powerful servers we recommend a lower value, and for slow servers the use of a higher value.

Please note that this delay won't slow down the publishing process, since it will be used for publishing dates only. 

Examples

DelayInSeconds=5
