DatabaseImplementation

Summary

Sets the type of database you are using.

Usage

 
DatabaseImplementation=ezmysql|ezpostgresql|ezoracle

Description

Set this option to "ezmysql" if you are using a MySQL database. If you are using PostgreSQL set this option to "ezpostgresql". In case you are using an Oracle database (note that the eZ Publish Extension for OracleÂ® Database is required for this) you should specify "ezoracle" in the "DatabaseImplementation" directive.
