DisplayWarnings

Summary

Allows you to display template engine warnings.

Usage

DisplayWarnings=enabled | disabled

Description

This setting sets whether warnings from the template engine should be displayed on the page or not.  Keep in mind that debug warnings are always shown if debugging is enabled.
This setting is case sensitive.

Examples

DisplayWarnings=enabled

Enabled by default
