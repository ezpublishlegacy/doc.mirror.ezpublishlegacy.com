rot13

Summary

Returns a ROT13 transformation of the input string.

Usage

input|rot13()

Returns

A rotated version of the input string.

Description

Returns a ROT13 transformation of the input string.

Examples

Example 1

{'Hello World'|rot13()}

The following output will be produced: "Uryyb Jbeyq".

Example 2

{'Uryyb Jbeyq'|rot13()}

The following output will be produced: "Hello world".
