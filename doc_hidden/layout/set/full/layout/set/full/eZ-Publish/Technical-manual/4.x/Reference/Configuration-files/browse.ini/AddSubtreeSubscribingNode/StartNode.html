StartNode

Summary

This setting sets the node from where the browsing should start when the action "AddSubtreeSubscribingNode" is used.

Usage

 
StartNode=<node>

Description

This setting sets the node from where the browsing should start when the action "AddSubtreeSubscribingNode" is used.
Nodes are used as value, such as content, users, media and setup.
This setting is case sensitive.

Examples

StartNode=content

"content" is the default value for this setting.
