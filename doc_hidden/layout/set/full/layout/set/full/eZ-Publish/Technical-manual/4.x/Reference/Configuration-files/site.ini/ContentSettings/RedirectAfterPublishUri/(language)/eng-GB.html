RedirectAfterPublishUri

Summary

Sets the redirect uri for RedirectAfterPublish (5.2 and higher)

Usage

RedirectAfterPublishUri=[uri]

Description

Sets the redirect uri to be considered by the RedirectAfterPublish setting for redirecting after a content publish.

If RedirectAfterPublish=uri, indicates to which URI user will be redirected after publishing a node.

If empty the parentNode value for RedirectAfterPublish will be considered for the redirect.

Important: This setting is only available as of eZ Publish 5.2

Examples

RedirectAfterPublish=uri
RedirectAfterPublishUri=/content/dashboard
