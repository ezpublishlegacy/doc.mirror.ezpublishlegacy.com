Requirements

eZ Image Editor LS 5.0.0 has the following software requirements to be able to run properly:

  • eZ Publish 5.0 and higher
  • eZ JS Core LS 5.0.0 shipped with eZ Publish 5.0 and higher
  • PHP 5.2 and higher + PHP GD2 or
  • PHP 5.2 and higher + ImageMagick

Technical information

eZ Image Editor uses the following software components and plug-ins:

  • jQuery (http://www.jquery.com/)
  • jQuery UI (http://www.jqueryui.com/)
  • Some jQuery plug-ins:
    • colorpicker (http://www.eyecon.ro/colorpicker/)
    • Jcrop (http://deepliquid.com/content/Jcrop.html)
  • PHP (http://www.php.net)
  • GD2 (http://www.php.net) or ImageMagick (http://www.imagemagick.org/)
  • eZ Components (mainly ImageConversion) (http://www.ez.no/)
