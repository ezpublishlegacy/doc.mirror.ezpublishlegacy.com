Repsitory

Summary

Sets the default location of the icon repository.

Usage

Repository=<path_to_location>

Description

This setting sets the default location of the icon repository.  It tells your installation where to look for icons which are relevant to eZ Publish.

This setting is case sensitive.

Examples

Repository=share/icons

This is the default folder containing eZ Publish related icons.
