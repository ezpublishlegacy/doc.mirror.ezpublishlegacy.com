eZ Survey 2.x

Creating a survey is a straight forward task through the use of eZ Survey 2.1, the survey tool for eZ Publish. Based on the original eZ Survey 1.0, its purpose is to create and publish surveys in a similar way as you would publish any content object.

eZ Survey 2.1 has many advantages. By creating a survey content class with the survey datatype provided by eZ Survey you can create surveys as you would an article, folder, image, etc... This survey class can of course be reused. Furthermore you will be able to add survey attributes to customize your survey, so that users will for example be able to give a score or add text. Also a Survey-tab is added to your Administration Interface for easy management of your surveys. And once you have results, eZ Survey enables you to export them as csv-files, statistics, etc...

Since the previous eZ Survey version (2.0) some changes have been made. Amongst other things the following things have been updated:

  • An installer was added.
  • When pressing the Browse-button in Related object configuration, storing of contentclass relation was added.
  • More information was added to the templates for questions that are not valid.
  • A description was added to the package to create a warning.
  • The name of the node was added to the subject field of the confirmation email.
  • In the CSV export a field was added for the user that gave an answer to the survey
  • Create xml with DOMDocument instead ezxml.
  • Added support for redirecting to an external url after submit a survey.
  • Added sql update script from 1.0 (mysql only)
  • Added compatibility with Oracle and Postgres
  • For bug fixes visit the Changelogs
