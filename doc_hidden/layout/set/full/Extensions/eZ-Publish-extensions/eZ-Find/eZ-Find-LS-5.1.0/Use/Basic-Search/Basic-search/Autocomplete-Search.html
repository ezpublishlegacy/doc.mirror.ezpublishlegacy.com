Autocomplete Search

To increase search usability, an auto-complete search function is available since eZ Find 2.3. This feature provides users with a list of suggested search words once they have entered a minimum amount of letters from a word in the search field.

When a user enters letters in the search field the search engine will try to predict the word the user wants to type without the user actually needing to type it in completely. The user can select the correct word from the suggestion list. As shown in the next screenshot.

The autocomplete function in the User Interface

When enabling the auto-complete function while working with eZ Flow, the search box will also provide this enhanced feature. The auto-complete function works both in the User Interface as well as in the Admin Interface search fields. The following screenshot shows an example in the Admin Interface:

The autocomplete function in the Administration Interface

The amount of letters required before the auto-complete function is activated can be configured with the AutoCompleteSettings in "ezfind.ini", but by default two letters will suffice. Please visit the chapters Configuration settings eZ Find and Auto-complete search function for more information on how to do this.
