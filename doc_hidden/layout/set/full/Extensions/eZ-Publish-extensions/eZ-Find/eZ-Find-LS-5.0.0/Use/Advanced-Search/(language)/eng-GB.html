Advanced Search

Where the basic search might be similar to other search engines, the advanced search is not. The advanced search enables you to further limit search conditions and can be accessed by clicking the "Advanced Search" link on the search result page, or linking to "http://<your.domain>/content/advancedsearch/". The “Search the exact phrase” field provides the same functionality as the phrase search in the basic search.
