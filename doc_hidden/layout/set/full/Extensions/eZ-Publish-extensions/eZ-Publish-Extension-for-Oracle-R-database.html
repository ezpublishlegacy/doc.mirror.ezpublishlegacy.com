eZ Publish Extension for Oracle® database

The eZ Publish Extension for Oracle® Database (also known as "ezoracle" extension) makes it possible to either use Oracle as a database server for eZ Publish or to connect to an external Oracle database using the standard eZ Publish database abstraction layer.

It makes use of the PHP Oracle functions. eZ Publish cluster mode is supported.

Note: As an Enterprise-customer you are adviced to install the eZ Network client before you install the eZ Oracle extension. This is to avoid error messages that the eZ Network client is not installed.

The source code and packaged versions for download can be found at http://projects.ez.no/ezoracle
