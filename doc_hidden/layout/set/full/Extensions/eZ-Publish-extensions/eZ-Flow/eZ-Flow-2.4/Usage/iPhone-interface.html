iPhone interface

eZ Flow comes with an interface for users that want to access their portal web-site via an iPhone. This is an out-of-the-box enabled site-access that gives users an easy way of reading portal information updates on their web-sites through a dedicated iPhone interface.

The interface is a dedicated site-access as well as design folder with iPhone optimized templates, style-sheets, etc.

  •  Accessing the iPhone-enabled site
  •  Go to another website
  •  Google search
  •  Switching between iPhone mode and full version mode
