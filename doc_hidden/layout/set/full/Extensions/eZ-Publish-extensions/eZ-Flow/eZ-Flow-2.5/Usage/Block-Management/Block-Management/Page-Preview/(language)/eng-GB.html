Page Preview

Please note that eZ Flow datatype is not entirely compatible with eZ Publish page preview mechanism. Only custom blocks (mainly template based) will be displayed properly in the preview window. Manual and dynamic blocks require a cronjob process to be executed in order to handle time based operations for those blocks. However, eZ Flow comes with a Timeline feature, which allows Editors to view a page within a certain time. This dedicated preview page can process dynamic and manual blocks and display their content.
