TemplateFunction

Summary

Sets if a template is to be executed instead of a php class function.

Usage

TemplateFunction=true | false

Description

This is an optional setting, which will define if a template is to be executed instead of a php class function. If set to true, then request will go to:

.../templates/<Class>/<function>.tpl

This setting is case sensitive.

Examples

Example
TemplateFunction=true
