LocalScriptBasePath

Summary

This setting maps the libraries to their local base script path.

Usage

 LocalScriptBasePath[<library>]=<location>

Description

This setting maps the libraries to their local base script path, which is needed for yui scripts (YUILoader).

This setting is case sensitive.

Examples

Example
LocalScriptBasePath[yui3]=lib/yui/3.1.1/build/
LocalScriptBasePath[yui2]=lib/yui/2.8.1/build/
