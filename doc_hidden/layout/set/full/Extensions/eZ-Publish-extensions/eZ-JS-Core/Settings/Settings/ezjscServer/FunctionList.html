FunctionList

Summary

This setting lists the permission functions.

Usage

FunctionList[]=<function>

Description

This setting lists the permission functions that are used by the eZ Publish permission system.

This setting is case sensitive.

Examples

Example
FunctionList[]=ezjsctemplate
FunctionList[]=ezjsckeyword
FunctionList[]=ezjscrating_rate
