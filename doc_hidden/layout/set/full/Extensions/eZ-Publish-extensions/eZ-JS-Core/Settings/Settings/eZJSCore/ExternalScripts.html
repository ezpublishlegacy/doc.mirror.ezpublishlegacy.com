ExternalScripts

Summary

This setting maps external JavaScript libraries to their CDN URL.

Usage

ExternalScripts[<library>]=<URL>

Description

This setting lists the external JavaScript libraries to use and maps them to their CDN URL. This setting needs to be set when the setting LoadFromCDN is set to enabled.

This setting is case sensitive.

Examples

Example
[ezjscore]
LoadFromCDN=enabled
ExternalScripts[yui3]=http://yui.yahooapis.com/3.1.1/build/yui/yui-min.js
ExternalScripts[yui2]=http://yui.yahooapis.com/2.8.1/build/yuiloader/yuiloader-min.js
ExternalScripts[jquery]=http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js

The system is allowed to load from external servers. The location of the external JavaScript libraries to use are listed under the setting ExternalScripts.
