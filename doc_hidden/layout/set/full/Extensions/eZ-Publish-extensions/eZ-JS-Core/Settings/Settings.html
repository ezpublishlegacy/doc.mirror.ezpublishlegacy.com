Settings

[eZJSCore]

SettingSummary
ExternalScripts This setting maps external JavaScript libraries to their CDN URL.
LoadFromCDN Setting to enable or disable the ability to load files from external servers.
LocalScriptBasePath This setting maps the libraries to their local base script path.
LocalScripts This setting maps the local JavaScript libraries to their location.
Packer Setting to enabled or disables the use of the js / css packer.
PreferredLibrary This setting defines the preferred library.

SettingSummary

SettingSummary

Function This setting lists the functions the user needs to have access to.
PermissionPrFunction Sets permissions per function.
TemplateFunction Sets if a template is to be executed instead of a php class function.
