Q: Where do I find the eZ Publish re-write rules with examples?

A: The eZ Publish re-write rules are found in the eZ Publish "Technical manual" in the Installation chapter under "Virtual host setup". The examples can be found in the chapter "Virtual host example". Links to the re-write rules with examples for each version of eZ Publish is found in the list below:

  • Re-write rules for eZ Publish 4.5 - "Virtual host example 4.5"
  • Re-write rules for eZ Publish 4.4 - "Virtual host example 4.4"
  • Re-write rules for eZ Publish 4.3 - "Virtual host example 4.3"
  • Re-write rules for eZ Publish 4.x - "Virtual host example 4.x"

Note: We recommend that you start your installation process with the re-write rules specification. Read more in the "Installation" chapter. Find a short description of the installation process in the FAQ - "How do I install eZ Publish?"
